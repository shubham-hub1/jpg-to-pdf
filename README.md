# JPG → PDF Converter

A lightweight, fully client-side image-to-PDF converter. No server, no uploads, no backend — everything runs in the browser.

## Features

- **Drag & Drop** — drop images directly onto the page or browse to select
- **Multi-image support** — combine multiple JPG / PNG / WEBP files into a single PDF
- **Reorder pages** — drag cards to rearrange before converting
- **Page settings** — choose page size (A4, Letter, A3, or fit to image), orientation (auto, portrait, landscape), margin, and quality
- **Custom filename** — name your output before downloading
- **100% client-side** — your images never leave your device

## Demo

Open `index.html` in any modern browser. No installation needed.

## Usage

1. Drop images onto the upload zone or click **browse files**
2. Drag cards to set the page order
3. Configure page size, orientation, margin, and quality
4. Enter an output filename
5. Click **Convert & Download**

## Tech Stack

- Vanilla HTML / CSS / JavaScript

Works in all modern browsers (Chrome, Firefox, Edge, Safari).
